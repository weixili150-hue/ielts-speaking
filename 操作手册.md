# ielts-speaking 项目操作手册

## 打开项目

```bash
cd /Users/liweixi/ielts-speaking && claude
```

一行命令：进入项目目录并启动 Claude Code。

## 本地运行测试

```bash
cd /Users/liweixi/ielts-speaking && npm start
```

然后浏览器打开 http://localhost:3457/app.html

## 部署到线上

代码推到 GitHub 后 Railway 自动部署，不需要手动操作：

```bash
cd /Users/liweixi/ielts-speaking
git add .
git commit -m "描述你改了什么"
git push origin main
```

等 1-2 分钟，访问 https://ielts-speaking-production.up.railway.app/app.html

## 更新 API Key

如果 DeepSeek key 换了，去 Railway 网页控制台 → ielts-speaking → Variables → 编辑 DEEPSEEK_API_KEY，保存后自动重新部署。

同时更新本地的 `.env` 文件。

## 项目文件

| 文件 | 作用 |
|------|------|
| server.js | 后端，AI 对话/评估/翻译接口 |
| public/app.html | 前端，整个口语网页 |
| package.json | 依赖 |
| CLAUDE.md | 给 Claude Code 看的项目说明书 |
